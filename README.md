# JP Meeting Caption Translator

A Chrome Extension for real-time translation of Google Meet captions. Supports Japanese, English, Vietnamese, Chinese, and Korean captions — with instant overlay translation using Google Translate (free) or OpenAI API.

---

## Overview

- **Live caption translation**: Instantly translates captions as they appear in Google Meet
- **Multi-language**: Detects and translates Japanese, English, Vietnamese, Chinese, and Korean
- **Overlay window**: Floating, draggable, scrollable translation history — never lose context
- **No backend/server**: All processing is done locally in your browser
- **Free by default**: Uses Google Translate API (no key required); OpenAI API optional for higher quality
- **Configurable**: Choose source/target language, provider, and model in Settings

---

## Quick Start

1. Build the extension (`npm install && npm run build`)
2. Load the `dist/` folder in Chrome as an unpacked extension
3. Open Google Meet, enable captions, and start translating
